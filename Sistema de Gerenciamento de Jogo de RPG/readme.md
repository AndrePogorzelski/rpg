# Sistema de Gerenciamento de RPG

Sistema web completo para gerenciamento de personagens de RPG desenvolvido em PHP procedural.

## Funcionalidades

### Básicas
- ✅ CRUD completo de personagens
- ✅ Cálculo automático de vida máxima
- ✅ Interface responsiva com Bootstrap
- ✅ Segurança básica (prevenção SQL injection)
- ✅ Tratamento de erros

### Pontuação Adicional
- ✅ Sistema de batalha entre personagens
- ✅ Upload de imagem para personagens
- ✅ Sistema de login de usuários
- ✅ Ranking de personagens por nível
- ✅ Exportação de dados para CSV
